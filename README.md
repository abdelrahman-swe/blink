# Blink E-commerce Website

Blink is a high-performance, modern e-commerce web application built with the latest web technologies. It provides a seamless shopping experience with features like real-time data fetching, internationalization, and a highly responsive user interface.

**Live Demo:** [https://blink-code-clouders.netlify.app/en/home].

## 🚀 Core Technologies

- **Framework:** [Next.js] (App Router)
- **Language:** [TypeScript]
- **Styling:** [Shadcn UI] & [Tailwind CSS 4]
- **State Management:** [Zustand] & [TanStack Query]
- **Form Handling:** [React Hook Form] & [Zod]
- **Internationalization:** Multi-language support (English & Arabic)

## ✨ Features

- ** Complete E-commerce Flow:** Browse products, manage cart, and navigate through a secure checkout process.
- ** Internationalization:** Fully localized interface supporting multiple languages.
- ** Dark Mode Support:** Seamlessly switch between light and dark themes.
- ** Authentication:** Secure login, registration, and password reset flows.
- ** OTP Verification:** Verify phone number / Emial Address with OTP.
- ** Advanced Search & Filtering:** Find products easily by category, brand, or specific search queries.
- ** User Profile Management:** Dedicated dashboard for account settings, security, and order history.
- ** User Orders:** View and track all user orders.
- ** Order & Return Tracking:** Stay updated on order statuses and manage returns directly from the profile.
- ** Favorites/Wishlist:** Save products to your personal list for quick access later.
- ** Payment Methods:** Paymob & COD
- ** Guest Checkout:** Checkout without registration (Paymob)
- ** Responsive Design:** Optimized for all devices, from mobile phones to desktops.

## 💎 Benefits

- **Blazing Fast Performance:** Leverages Next.js Server Components and optimized asset loading for near-instant transitions.
- **Seamless User Experience:** Real-time UI updates with TanStack Query and smooth states.
- **Type Safety:** Built with TypeScript to ensure code quality and reduce runtime errors.
- **Scalable Architecture:** Modular component design and robust state management make it easy to extend and maintain.
- **Modern UI/UX:** A premium look and feel inspired by modern design trends.
